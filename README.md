# pets-project
Site de animais para estudo de Node e Back-end

Projeto feito com Node + Typescript


### Pré-requisitos globais:
`npm i -g nodemon typescript ts-node`

### Instalação
`npm install`

### Parad rodar o projeto
`npm run start-dev`
